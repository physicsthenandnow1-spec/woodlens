Google Drive Setup Guide
One-time setup (~5 minutes). Do this on a desktop browser first.
1. Create a Google Cloud Project
Go to console.cloud.google.com → New Project → Name it WoodLens → Create.
2. Enable Google Drive API
APIs & Services → Library → search Google Drive API → Enable.
3. Configure OAuth Consent Screen
APIs & Services → OAuth consent screen → External → Create.
App name: WoodLens · Add your Gmail as a Test user.
Scopes: add drive.readonly and drive.file → Save.
4. Create OAuth Client ID
Credentials → Create Credentials → OAuth client ID.
Type: Web application.
Authorized JS origins: https://yourname.github.io
→ Create → Copy the Client ID.
5. Paste into WoodLens
DRIVE tab → paste Client ID → SAVE → CONNECT DRIVE.
